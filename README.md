# BastionHost_Project
A project simulating a Bastion host (Jumping server) in AWS using EC2

-- 📐 Architecture --
1- Two Diferents VPCs (Diferents IPV4 CIDR)
2- VPC peering
3- Subnets
4- ACLs for minimun traffic 
5- EC2 machines 
6- Security Groupes

-- 🛠️ Technologies and Services --
1- AWS EC2
2- AWS VPC
3- VPC Peering
4- ACLs and Security Groups
